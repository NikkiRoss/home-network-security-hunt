Home Network Security Hunt

Project Overview

This project documents a beginner-level security investigation of an authorized home network. The goal is to identify connected devices, discover exposed network services, evaluate potential risks, and recommend security improvements.

Objectives

- Identify devices connected to the network
- Determine which ports and services are exposed
- Investigate unexpected devices or services
- Evaluate potential security risks
- Recommend appropriate remediation steps
- Document the investigation using a professional security-report format

Scope and Authorization

This investigation is limited to my own home network and devices that I own or have permission to examine.

No systems outside the authorized network will be scanned or tested.

Tools

The project may use:

- Nmap
- Windows Command Prompt or PowerShell
- Wireshark
- Router administration interface
- Git and GitHub

Investigation Process

1. Define the authorized scope
2. Record the network configuration
3. Discover connected devices
4. Scan authorized devices for open ports
5. Research detected services
6. Identify possible security concerns
7. Apply or recommend remediation
8. Verify improvements
9. Document findings

Safety and Privacy

Sensitive information will be removed before publication, including:

- Public IP addresses
- Full private IP addresses when unnecessary
- MAC addresses
- Wi-Fi names and passwords
- Router login information
- Device serial numbers
- Names or personal information belonging to household members
